# Markdown-it Plugin for TemplateMark

This package extends CommonMark for TemplateMark to introduce three new DOM nodes:
1. Clause
2. Variable
3. ComputedVariable

## Installation

```sh
npm install @accordproject/markdown-it-template --save
```
